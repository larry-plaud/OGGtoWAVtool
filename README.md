## 📖 简介 | Introduction

**OGG to WAV Converter** 是一款专为批量处理音频文件而设计的桌面端转换工具。它可以帮助用户快速将指定文件夹内的所有 `.ogg` 音频文件无损转换为 `.wav` 格式。

无论是处理海量的测试数据（如 RF Test 数据）、游戏音效，还是日常音频格式转换，本工具都能为您节省大量时间。

## ✨ 核心特性 | Features

  * **📦 批量处理**：一键导入整个文件夹，自动识别并转换所有 OGG 文件。
  * **📂 保留目录结构**：支持“保留子目录结构”功能，转换后的输出文件将完美复现原有的文件夹层级，告别混乱。
  * **⚡ 高效转换**：多文件并发处理，实时显示转换日志与整体进度条。
  * **🛠 可选覆盖**：提供“覆盖已有文件”选项，方便进行重复测试或数据更新。
  * **🖥 直观界面**：极简的 GUI 设计，操作一目了然，无需任何命令行基础。

## 🚀 快速开始 | Quick Start

### 1\. 下载与运行

进入 [Releases](https://www.google.com/search?q=%E4%BD%A0%E7%9A%84Releases%E9%93%BE%E6%8E%A5) 页面，下载最新版本的执行文件（例如 `OggToWavConverter.exe`）。解压后双击即可运行，无需繁琐的安装过程。

### 2\. 使用步骤

1.  **添加源文件**：点击界面左下角的 **`+ 添加文件夹`** 按钮，选择包含 `.ogg` 文件的根目录。
2.  **设置输出目录**：点击界面右上角的 **`浏览...`** 按钮，选择转换后 `.wav` 文件存放的目标文件夹。
3.  **配置选项（可选）**：勾选 **`保留子目录结构`** 以维持原有的文件分类。
4.  **开始转换**：点击右下角的 **`▶ 开始转换`**，观察右侧日志窗口即可掌握实时进度。
